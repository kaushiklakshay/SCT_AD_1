# Android Calculator App
A sleek, intuitive, and fully functional calculator application built for Android. This app handles basic arithmetic operations with a clean user interface, making everyday calculations quick and effortless.

### 🚀 Features
* Standard arithmetic operations (+, -, *, /)
* Instant result updates
* Error handling (e.g., preventing division by zero)
* Clean and modern user interface

### 🛠️ Built With
* **Language:** html/Java/css
* **Development Environment:** Android Studio
* **Minimum SDK:** Android 5.0 (API 21)

### 📲 How to Try It
You can download the compiled APK directly from the repository files to install and test the app on your Android device.
